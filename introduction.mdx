## What is OmniSearches?

OmniSearches is a powerful AI search engine that combines the capabilities of Google's Gemini 2.0 Flash and Deepseek's reasoning models to deliver intelligent, source-backed answers to your queries. Built with modern web technologies and designed for both developers and end users, OmniSearches provides real-time information with verifiable sources.

## How It Works

OmniSearches operates through a sophisticated pipeline:

1. **Query Processing**: Your search query is analyzed to determine the optimal search strategy
2. **Web Grounding**: Real-time searches are performed using Google Search integration
3. **AI Analysis**: Gemini 2.0 Flash processes search results to generate intelligent summaries
4. **Source Attribution**: Every claim is linked back to its source for verification
5. **Enhancement**: Related questions and relevant images are added to enrich your results

## Technology Stack

OmniSearches is built on modern, proven technologies:

- **Frontend**: React + Vite + TypeScript + Tailwind CSS
- **Backend**: Express.js + TypeScript
- **AI Models**: Google Gemini 2.0 Flash API + OpenRouter Deepseek
- **Search**: Google Search API integration
- **Images**: Wikimedia Commons API
